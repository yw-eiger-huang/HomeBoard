# 木板構型幾何規格文件

> 版本：1.0 | 座標系：2D 側視圖 | 單位：公尺（除非特別標註）

---

## 1. 基本構型描述

一塊木板以單點 hinge（A 點）固定於天花板，木板**外傾**（底端朝牆方向斜下），底端在設計角度（40°）時剛好接觸牆面。木板底部 65 cm 為可折疊段，以 hinge 繞 B 點旋轉，摺疊方向僅限朝牆面方向（往內折）。

## 4. 幾何關鍵點定義

所有坐標以**靠牆側邊緣**為基準（除 E F G 外）。

| 點 | 名稱 | 說明 | 是否隨角度變動 |
|----|------|------|----------------|
| **A** | Hinge | 天花板固定點，靠牆側邊緣 | 固定 |
| **B** | 折點 | 主板底端 = 摺疊段頂端，靠牆側邊緣，摺疊旋轉軸 | 隨牆面夾角變動 |
| **C** | 主板底端 | 木板全長末端（未折疊位置），靠牆側邊緣 | 隨牆面夾角變動 |
| **D** | 摺疊端 | 摺疊段末端，靠牆側邊緣 | 隨牆面夾角＋摺疊角度變動 |
| **E** | AB 四等分 1/4 | 主板中心線 1/4 處 | 隨牆面夾角變動 |
| **F** | AB 四等分 2/4 | 主板中心線 2/4 處（中點） | 隨牆面夾角變動 |
| **G** | AB 四等分 3/4 | 主板中心線 3/4 處 | 隨牆面夾角變動 |
| **H** | 牆角 | 天花板與牆面交點 (0, 2.5) | 固定 |
| **I** | 全長觸牆點 | angle=40° 時 C 點位置（觸牆參考點） | 固定 |

## 5. 座標計算公式

### 5.1 輸入變數

| 變數 | 範圍 | 說明 |
|------|------|------|
| `θ`（angle） | 0° ~ 40° | 木板靠牆側邊緣與牆面的夾角；0° = 垂直貼牆，40° = 設計角度 |
| `φ`（foldAngle） | −90° ~ 0° | 摺疊段相對主板方向的旋轉角；0° = 展開共線，負值 = 往牆折 |

### 5.2 方向向量

```
主板方向向量：  dir  = (−sin θ, −cos θ)
主板外法向量：  norm = (+cos θ, −sin θ)   ← 垂直板面，指向遠離牆側

摺疊方向向量：  fold_dir  = rotate(dir,  φ)
摺疊外法向量：  fold_norm = rotate(norm, φ)
```

### 5.3 靠牆側邊緣座標

```
A = (AX,  AY)                      AX = 3.048 × sin(40°) ≈ 1.9593 m
                                    AY = 2.500 m

B = A + 2.398 × dir
C = A + 3.048 × dir
D = B + 0.650 × fold_dir
```

### 5.4 外側邊緣座標（板厚偏移）

```
A_outer = A + 0.05 × norm
B_outer = B + 0.05 × norm
C_outer = C + 0.05 × norm

摺疊段外側上角：  B_fold_outer = B + 0.05 × fold_norm
摺疊段外側下角：  D_outer      = D + 0.05 × fold_norm
```

### 5.5 中心線座標（板厚 / 2 偏移）

```
A_cen = A + 0.025 × norm
B_cen = B + 0.025 × norm
E = A_cen + (1/4) × 2.398 × dir
F = A_cen + (2/4) × 2.398 × dir
G = A_cen + (3/4) × 2.398 × dir
```

### 5.6 固定參考點

```
H = (0, 2.500)
I = (0, 2.500 − 3.048 × cos(40°))  ≈ (0, 0.165 m)
```

## 6. 板面多邊形（繪製用）

### 主板（四邊形）
```
頂點順序：A → B → B_outer → A_outer → (close)
```

### 摺疊段（四邊形，剛性長方形）
```
頂點順序：B → D → D_outer → B_fold_outer → (close)
```

> 摺疊段繞 B 點旋轉時整體為剛性長方形，不形變。`B_fold_outer` 隨 `fold_norm` 旋轉，與主板外側 `B_outer` 不重合（除 φ = 0° 時）。

## 8. 角度約束條件

### 牆面夾角（θ）
- 範圍：**0° ~ 40°**
- 0° 極限：木板垂直貼牆（B 與 C 均在牆面上）
- 40° 極限：設計角度，C 觸牆

### 摺疊角度（φ）
- 範圍：**−90° ~ 0°**（僅允許往牆折）
- 0°：展開，D = C（摺疊段與主板共線）
- 下限由幾何約束動態計算，確保：
  - D.x ≥ 0（不穿越牆面）
  - D.y ≥ 0（不低於地面）
  - D.y ≤ 2.5（不高於天花板）
