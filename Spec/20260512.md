# 木板構型幾何規格文件

> 版本：5.0 | 座標系：3D 斜平行投影（cabinet oblique）+ Y 軸旋轉 | 單位：公尺（除非特別標註）

---

## ── 基礎定義 ──

## 1. 基本構型描述

一塊木板以單點 hinge（A 點）固定於天花板，木板**外傾**（底端朝牆方向斜下）。**D 點（摺疊端）恆固定於 I = (0, 0.165)**；主板角度 θ 改變時，A 點沿天花板朝牆角 H 滑動，折疊段自動旋轉以保持 D 在 I。木板底部 65 cm 為可折疊段，以 hinge 繞 B 點旋轉。**摺疊方向及角度無 UI 控制，由 D=I 幾何約束自動決定。**

G 點為主板上的定位點，沿板方向距 A 點 `gOffset`，隨板移動；H 點固定於牆面-天花板交角 (0, CEIL_H)，恆不動。主板與折疊段以 3D 矩形柱體渲染；2D 模式只繪製 z=0 那側。

## 2. 座標系與邊界

| 元素 | 定義 |
|------|------|
| 原點 (0, 0, 0) | 牆面與地面交點（正面） |
| +X | 遠離牆面方向（水平） |
| +Y | 向上（垂直） |
| +Z | 朝觀察者方向（正面 = Z=0，背面 = Z=BOARD_W） |
| 牆面 | x = 0 |
| 地面 | y = 0 |
| 天花板 | y = CEIL_H = 2.500 m |

## 3. 固定參數

| 常數 | 值 | 說明 |
|------|----|------|
| `BOARD_LEN` | 3.048 m | 板全長（含摺疊段） |
| `FOLD_LEN` | 0.650 m | 摺疊段長（B→D） |
| `MAIN_LEN` | 2.398 m | 主板段長（A→B）= BOARD_LEN − FOLD_LEN |
| `T` | 0.018 m（1.8 cm） | 面板厚度（外側面） |
| `T_FRAME` | 0.045 m（4.5 cm） | 組合總深 = T + 0.027 m |
| `HT` | T_FRAME / 2 = 0.0225 m | 截面中心線偏移 |
| `BOARD_W` | 3.660 m | 板寬（Z 軸深度） |
| `CEIL_H` | 2.500 m | 天花板高 |
| `AX_W` | 1.9593 m | A 點在 θ=40° 時的離牆距離（= 3.048 × sin 40°；用於場景縮放基準） |

A、G、B 鉸接點位於外側面（距內側面 `T_FRAME` 處）。所有外側邊點（Ao、Bo、Bfo、Do）以 `T_FRAME × 法向量` 從內側邊偏移。

---

## 4. 幾何關鍵點定義

所有坐標以**靠牆側邊緣（內側，w 字尾）**為基準（含 G）。H、I、D 為固定點；A、B 隨 θ 變動。

| 點 | 名稱 | 說明 |
|----|------|------|
| **Aw** | Hinge | 天花板滑動點，靠牆側邊緣；θ↓ 則 A.x 朝 H 靠攏 |
| **Bw** | 折點 | 主板底端 = 摺疊段頂端，靠牆側邊緣 |
| **Dw** | 摺疊端（固定） | 恆 = I = (0, CEIL_H − BOARD_LEN × cos 40°) ≈ (0, 0.165) |
| **Gw** | G 定位點 | 主板靠牆側邊緣上，距 Aw 點 `g`（可調）之點；隨板移動 |
| **H** | 牆角 | 天花板與牆面交點 (0, 2.5)，固定不動 |
| **I** | 全長觸牆點 | θ=40° 時 Cw 的位置（觸牆參考點），固定；= Dw 恆成立 |

## 5. 座標計算公式

### 5.1 輸入變數

| 變數 | 範圍 | 說明 |
|------|------|------|
| `θ`（angle） | 0° ~ 40° | 木板靠牆側邊緣與牆面的夾角；0° = 板垂直貼牆，40° = 設計角度 |
| `g`（gOffset） | 1.75 ~ 2.15 m | Gw 點距 Aw 點沿板邊距離（使用者可調） |

### 5.2 方向向量

```
dir      = (−sin θ, −cos θ)              ← 主板方向（Aw→Bw）
norm     = (+cos θ, −sin θ)              ← 主板外法向量（遠離牆側）
boardDir = atan2(−cos θ, −sin θ)
```

### 5.3 靠牆側邊緣座標（inner edge）

D 固定於 I；B.y 由主板角度決定；B.x 由 |B→D| = FOLD_LEN 約束反算；A 由 B 導出：

```
Dw = (0, CEIL_H − BOARD_LEN × cos 40°)  ← 固定於 I ≈ (0, 0.165)

Bw.y = CEIL_H − MAIN_LEN × cos θ
dy_BD = Dw.y − Bw.y
Bw.x  = √(FOLD_LEN² − dy_BD²)           ← |B→D|=FOLD_LEN 約束，B 恆在 D 右側

Aw   = (Bw.x + MAIN_LEN × sin θ,  CEIL_H)   ← 天花板上，θ↓ 時 Aw.x 朝 0 靠攏
Gw   = Aw + g × dir                          ← G 隨板移動
H    = (0, 2.500)                            ← 固定
I    = (0, CEIL_H − BOARD_LEN × cos 40°)    ← 固定，= Dw
```

### 5.4 摺疊段（D=I 自動同步）

```
fdx = (Dw.x − Bw.x) / FOLD_LEN     ← B→D 單位向量 x
fdy = (Dw.y − Bw.y) / FOLD_LEN     ← B→D 單位向量 y
foldDir = atan2(fdy, fdx)
fnx = −fdy,  fny = fdx              ← 摺疊段外法向量
```

### 5.5 外側邊緣（o 字尾）與中心線（c 字尾）

```
── 主板外側（+ norm × T_FRAME）──
Ao = Aw + T_FRAME × norm
Bo = Bw + T_FRAME × norm

── 摺疊段外側（+ fold_norm × T_FRAME）──
Bfo = Bw + T_FRAME × (fnx, fny)
Do  = Dw + T_FRAME × (fnx, fny)

── 主板中心線（+ norm × HT）──
Ac = Aw + HT × norm
Bc = Bw + HT × norm
```

## 8. 輔助量測

| 量測項 | 公式 |
|--------|------|
| H→G 距離 | `\|H − Gw\|` |
| H→B 距離 | `\|H − Bw\|` |
| A→B 水平投影 | `Aw.x − Bw.x` |
| B 離地 | `Bw.y` |
| D 離地 | `Dw.y`（固定 ≈ 16.5 cm） |
| 內縮基準點離地高度 | 見 §8.1 |

### 8.1 內縮基準點投影高度

使用者可調整「內縮距離」d（100～150 cm），以 Ao 的地面投影為基準往牆面方向內縮 d，向上投影至主板外側面，標示交點離地高度。

```
refFloorX = Ao.x − d
t         = (refFloorX − Bo.x) / (Ao.x − Bo.x)    （t ∈ [0,1] 時有效）
projY     = Bo.y + t × (Ao.y − Bo.y)
```

`t ∉ [0,1]` 時標示自動隱藏。**視覺元素（`showDims` 開啟時）：** 地面水平 dim 線、垂直灰虛線、交點灰色圓點、高度標籤（cm）。

## ── 3D 投影 ──

## 9. 投影規格

### 9.1 Cabinet Oblique 基本參數

```
投影角 30°，foreshortening 0.5×
OBL_COS = cos(30°) × 0.5 ≈ 0.4330
OBL_SIN = sin(30°) × 0.5 = 0.25
```

### 9.2 水平視角旋轉（Y 軸旋轉）

視角旋轉角 φ（`params.hViewAngle`，−90°～90°）繞 Y 軸旋轉整個場景：

```
cosV = cos(φ),  sinV = sin(φ)
h(x,z) = x·cosV + z·sinV
d(x,z) = −x·sinV + z·cosV

p(x, y, z):
  canvas_x = OX + h·S + d·S·OBL_COS
  canvas_y = OY − y·S − d·S·OBL_SIN
```

φ=0°：側視圖。**模式切換**：切換至 2D 時 φ 重設為 0°；切換回 3D 時還原（`savedHViewAngle`）。

### 9.3 場景縮放（中心旋轉軸）

```
sceneW = AX_W + T_FRAME + 0.5
sceneH = CEIL_H × 1.22
S      = min((W−2·margin)/sceneW, (H−2·margin)/sceneH) × viewScale

hcCenter0 = cx + cz·OBL_COS
hcCenter  = (cx·cosV + cz·sinV) + (−cx·sinV + cz·cosV)·OBL_COS
OX        = margin + viewOX + 44·dpr + (hcCenter0 − hcCenter)·S
```

## 15. 邊框標示（Border Mark）

### 15.1 邊框寬度參數

| 滑桿 ID | 參數鍵 | 範圍 | 預設 | 作用邊 |
|---------|--------|------|------|--------|
| `#borderMarkWidth` | `borderMarkWidth` | 3.6–10 cm | 5 cm | A 端、D 端、左右 Z 邊 |
| `#borderMarkJunctionWidth` | `borderMarkJunctionWidth` | 3–5 cm | 3 cm | B 交接端 |

### 15.2 幾何定義

設 `M = borderMarkWidth`，`J = borderMarkJunctionWidth`。

主板內框：上邊 = `AY_w − M×cos(θ)`，下邊 = `BY_w + J×cos(θ)`，左右 z = M / BOARD_W−M。
摺疊板內框：上邊 = `BY_w + J×fdy`，下邊 = `DY_w − M×fdy`，左右 z = M / BOARD_W−M。
3D 視圖：`ctx.fill('evenodd')`，depth − 0.002。

## 18. 2D 正面視圖

左欄側視、右欄正面（`drawFrontView()`），分隔線 `#3a4060` 於 `x = W/2`。

### 18.1 投影

正交，觀察方向 +X，投影面 Y-Z。`cvX(z) = fvOX + z·S`，`cvY(y) = margin + viewOY + (CEIL_H−y)·S`。

### 18.2 關鍵高度

```
AY_w = CEIL_H
BY_w = AY_w − MAIN_LEN × cos(θ)
DY_w = BY_w + FOLD_LEN × sin(foldDir)
```

### 18.3 圓孔投影位置

主板孔：`hy = AY_w − (BOARD_LEN − u) × cos(θ)`；折疊段孔：`hy = BY_w + (FOLD_LEN − u) × fdy`。

### 18.4 比例尺

`autoS = min(sideW/sceneW, frontW/BOARD_W, (H−margin×2)/sceneH)`，`frontW = W/2 − margin − 90·dpr`。

## 19. 摺疊板折疊範圍與隱藏規則

### 19.1 折疊角

折疊方向由 B→D=I 動態決定：`foldDir = atan2((Dw.y−Bw.y)/FOLD_LEN, (Dw.x−Bw.x)/FOLD_LEN)`。

θ=40° 時 foldDir ≈ boardDir（板共面）；θ 縮小時折疊段漸向牆側折回。

### 19.2 正面視圖隱藏條件

`fdy = sin(boardDir − foldRad)`。`fdy >= 0` 時**僅正面視圖**隱藏摺疊板相關元素。

臨界角：θ < 4050/145 ≈ **27.9°** 時隱藏；θ ≥ 27.9° 時顯示。

### 19.3 側視圖恆顯示

fdx 恆負，側視圖無條件繪製所有摺疊板元素。

## 25. 程式碼模組結構

| 檔案 | 職責 |
