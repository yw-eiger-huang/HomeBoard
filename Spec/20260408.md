## 16. 2D 正面視圖

### 16.1 觸發條件

切換至 2D 模式（`show3D = false`）時，canvas 分為左右兩欄：
- **左欄**（`x < W/2`）：既有側視圖，自動縮放比例限於左欄寬度。
- **右欄**（`x ≥ W/2`）：正面視圖，由 `drawFrontView()` 繪製。
- 分隔線：虛線，`x = W/2`，樣式 `#3a4060`。

### 16.2 投影方式

正交投影，觀察方向為主板外側面法向量（沿 `(mnx, mny, 0)` 方向）。

| 座標軸 | 方向 | 範圍 |
|--------|------|------|
| 橫軸 z | 板寬 Z | 0 → BOARD_W (3.66 m) |
| 縱軸 u | 距 D 端距離 | 0 (D) → BOARD_LEN (A) |

投影函式：`fv(z, u) = [fvOX + z·fvS, fvOY − (u − uMin)·fvS]`

**頂部對齊**：正面視圖板頂 A 的 canvas Y 與側視圖 A 點 canvas Y 對齊：`fvOY = margin + viewOY + bH_px`。

### 16.3 折疊段前視縮短

折疊段繞板面法向量在 X-Y 平面旋轉，正面視圖中沿縱軸被 `cos(foldRad)` 縮短：

```
foldRad = (2π/3) × (1 − angle/40)
D_u     = FOLD_LEN × (1 − cos(foldRad))   // D 點在正面視圖中的 u 座標
```

- `foldRad = 0`（angle=40）：D_u = 0，折疊段滿高顯示。
- `foldRad = π/2`（angle=10）：D_u = FOLD_LEN，折疊段壓縮為零。
- `foldRad > π/2`（angle<10）：D_u > FOLD_LEN，D 出現於 B 上方（折超過 90°）。

折疊段矩形頂點：`fv(0, D_u)` → `fv(BOARD_W, D_u)` → `fv(BOARD_W, FOLD_LEN)` → `fv(0, FOLD_LEN)`。

### 16.4 圓孔前視位置

主板圓孔（`u ≥ FOLD_LEN`）：位置不變，直接以 `fv(z, u)` 繪製。

折疊段圓孔（`u < FOLD_LEN`）：前視縮短後位置：

```
apparent_u = FOLD_LEN + (u − FOLD_LEN) × cos(foldRad)
```

開關同 §15.5（`showScrewHoles`）；`showFold = false` 時折疊段孔不繪製。

### 16.5 標注與標籤

| 元素 | 說明 |
|------|------|
| 板寬標注 | `BOARD_W = 3.66m`，水平標注線位於板底下方 `dimOff` |
| 主板高標注 | `MAIN_LEN`，縱向標注線於板右側 `dimOff` |
| 折疊高標注 | `FOLD_LEN`，縱向標注線於板右側 `dimOff×2`；`|D_u − FOLD_LEN| ≤ 0.02` 時隱藏 |
| 點標籤 | A（板頂中央）、B（左邊緣折線處）、D（折疊段末端中央） |
| 段落標籤 | `A→B 2.40m`（主板中央）、`B→D 0.65m`（折疊段中央） |
| 標題 | `正面視圖`，canvas 底部 |
| 側視圖標題 | `側視圖`，canvas 底部左欄 |

### 16.6 左欄縮放調整

2D 模式下側視圖 `autoS` 計算寬度限於左欄，需扣除標注偏移 `44·dpr`：

```js
sideW = params.show3D ? W − margin×2 : W/2 − margin − 44·dpr
autoS = min(sideW / sceneW, (H − margin×2) / sceneH)
```

## 15. 螺絲孔配置

### 15.1 座標定義

孔位以「距 D 端沿板長方向的距離 u」（公尺）及「沿板寬 Z 軸位置 z」描述。D 端為折疊段末端（u = 0），A 端為天花板端（u = 3.048 m）。

### 15.2 圓孔（255 個，17 × 15）

| 方向 | 起點 | 間距 | 數量 | 備註 |
|------|------|------|------|------|
| 橫向（Z） | 0.23 m | 0.20 m | 17 | 終點 3.43 m，左右各餘 0.23 m（對稱） |
| 縱向（U） | 0.10 m | 0.20 m | 15 | 終點 2.90 m，頂側餘 0.148 m |

- 孔徑：直徑 1.2 cm（半徑 0.6 cm）
- 與 B 點轉折線（u = 0.65 m）最近孔緣距離：**4.4 cm**，無重疊

### 15.3 格心點（224 個，16 × 14）

格心位於相鄰圓孔之間的格格中心：

| 方向 | 起點 | 間距 | 數量 |
|------|------|------|------|
| 橫向（Z） | 0.33 m | 0.20 m | 16 |
| 縱向（U） | 0.20 m | 0.20 m | 14 |

### 15.4 繪製規則

- **可見面判斷**：以 shoelace 法計算外側面投影帶號面積（`sArea`）；面積 < 0 → 外側面朝向視線（`isOuter=true`），否則內側面（`isOuter=false`）。
- **深度排序**：entry depth = 可見面 depth − 0.001，繪於該面之上（Painter's Algorithm）。
- **僅繪可見面**；不繪背向視線那面。
- **折疊段連動**：折疊段孔跟隨「摺疊段」開關；主板孔獨立受「圓孔」開關控制。
- **符號**：實心圓點，`rgba(255,255,255,0.80)`。
- **半徑**：`max(HOLE_R_M × S, 1.5 × dpr)`。

### 15.5 顯示開關

| 開關 ID | 參數鍵 | 預設 |
|---------|--------|------|
| `#togScrewHoles` | `showScrewHoles` | on |

- **2D 模式下強制隱藏**：渲染條件為 `show3D && showScrewHoles`；切換至 2D 時圓孔不顯示，`showScrewHoles` 值保持不變，切回 3D 後自動還原。

### 15.6 程式碼常數

```js
HOLE_R_M = 0.006   // 孔半徑（m）
SCREW_Z  = [0.23, 0.43, …, 3.43]   // 17 個 z 位置
SCREW_U  = [0.10, 0.30, …, 2.90]   // 15 個 u 位置
CELL_Z   = [0.33, 0.53, …, 3.33]   // 16 個格心 z
CELL_U   = [0.20, 0.40, …, 2.80]   // 14 個格心 u
```
