## 16. 2D 正面視圖

### 16.1 觸發條件

切換至 2D 模式（`show3D = false`）時，canvas 分為左右兩欄：
- **左欄**（`x < W/2`）：側視圖。
- **右欄**（`x ≥ W/2`）：正面視圖，由 `drawFrontView()` 繪製。
- 分隔線：虛線，`x = W/2`，樣式 `#3a4060`。

### 16.2 投影方式

正交投影，觀察方向沿 **+X 軸**（從室內看向牆面），投影面為 Y-Z 平面。

| 軸 | 含意 | 範圍 |
|----|------|------|
| 橫軸 Z | 板寬 | 0 → BOARD_W (3.66 m) |
| 縱軸 Y | 離地高度 | 0 (地板) → CEIL_H (天花板) |

Canvas 座標函式（與側視圖共用比例尺 `S`，A 點縱向自動對齊）：

```
cvX(z) = fvOX + z·S
cvY(y) = margin + viewOY + (CEIL_H − y)·S
```

`fvOX` 將板寬置中於右欄：`fvOX = W/2 + (W/2 − margin − BOARD_W·S) / 2`

### 16.3 關鍵高度

牆面夾角 θ（`params.angle`，弧度 `rad`）改變時，三個世界 Y 值均隨之更新：

```
AY_w = CEIL_H
BY_w = AY_w − MAIN_LEN × cos(θ)          // B 折線處
fdy  = −cos(θ + foldRad)                  // 折疊段 Y 方向分量
DY_w = BY_w + FOLD_LEN × fdy              // D 折疊末端
```

其中 `foldRad = (2π/3) × (1 − angle/40)`。

- `angle=40`：`BY_w ≈ 0.66 m`，`DY_w ≈ 0.16 m`（折疊段向下展開）。
- `angle=0`：`BY_w ≈ 0.10 m`，`DY_w > BY_w`（折超過 90°，D 出現於 B 上方）。

### 16.4 圓孔投影位置

| 區段 | 公式 |
|------|------|
| 主板孔（`u ≥ FOLD_LEN`） | `hy = AY_w − (BOARD_LEN − u) × cos(θ)` |
| 折疊段孔（`u < FOLD_LEN`） | `hy = BY_w + (FOLD_LEN − u) × fdy` |

開關同 §15.5（`showScrewHoles`）；`showFold = false` 時折疊段孔不繪製。

### 16.5 標注與標籤

| 元素 | 說明 |
|------|------|
| 板寬標注 | `BOARD_W = 3.66m`，水平標注線位於最低邊緣下方 `dimOff` |
| 主板高標注 | `MAIN_LEN`，縱向標注線於板右側 `dimOff` |
| 折疊高標注 | `FOLD_LEN`，縱向標注線於板右側 `dimOff×2`；投影高度 ≤ 5px 時隱藏 |
| 點標籤 | A（板頂中央）、B（左邊緣折線處）、D（折疊段末端中央） |
| 段落標籤 | `A→B 2.40m`（主板中央）、`B→D 0.65m`（折疊段中央） |
| 標題 | `正面視圖`，canvas 底部右欄 |
| 側視圖標題 | `側視圖`，canvas 底部左欄 |

### 16.6 比例尺與縮放

側視圖與正面視圖共用同一比例尺 `S = autoS × viewScale`。`autoS` 在 2D 模式下同時受三個約束：

```js
sideW  = W/2 − margin − 44·dpr
frontW = W/2 − margin − 90·dpr        // 右側留給標注線的空間
autoS  = min(sideW/sceneW, frontW/BOARD_W, (H − margin×2)/sceneH)
```

（§1–§15 均不變，以 v3.4/v3.5 為準。）

## 17. 程式碼模組結構

`board_viz.js` 依職責拆分為 ES Modules（`type="module"`），並以 `board_viz.js` 作為入口點 `import`：

| 檔案 | 職責 |
|------|------|
| `constants.js` | 所有物理/視覺常數與網格陣列（`BOARD_LEN`、`FS`、`SCREW_Z` 等） |
| `geometry.js` | `getGeom()`、`clampAngle()` |
| `state.js` | `canvas`、`ctx`、`params`、`view`（含平移/縮放/拖曳狀態） |
| `frontview.js` | `drawFrontView()` 及內部標注輔助函式 |
| `draw.js` | `draw()`、`drawFace3D()`、點標記輔助函式 |
| `events.js` | 所有滑桿、切換按鈕、滑鼠、觸控、視窗 resize 監聽器；`resize()` |
| `board_viz.js` | 入口點：`import { resize } from './events.js'; resize();` |

模組依賴方向（無循環）：`board_viz.js` → `events.js` → `draw.js` → `frontview.js` → `state.js`/`constants.js`；`geometry.js` → `state.js`/`constants.js`。

## 15. 螺絲孔配置

### 15.1 座標定義

孔位以「距 D 端沿板長方向的距離 u」（公尺）及「沿板寬 Z 軸位置 z」描述。D 端為折疊段末端（u = 0），A 端為天花板端（u = 3.048 m）。

### 15.2 圓孔（255 個，17 × 15）

| 方向 | 起點 | 間距 | 數量 | 備註 |
|------|------|------|------|------|
| 橫向（Z） | 0.23 m | 0.20 m | 17 | 終點 3.43 m，左右各餘 0.23 m（對稱） |
| 縱向（U） | 0.10 m | 0.20 m | 15 | 終點 2.90 m，頂側餘 0.148 m |

- 孔徑：直徑 1.2 cm（半徑 0.6 cm）
- 與 B 點轉折線（u = 0.65 m）最近孔緣距離：**4.4 cm**，無重疊

### 15.3 格心點（224 個，16 × 14）

格心位於相鄰圓孔之間的格格中心：

| 方向 | 起點 | 間距 | 數量 |
|------|------|------|------|
| 橫向（Z） | 0.33 m | 0.20 m | 16 |
| 縱向（U） | 0.20 m | 0.20 m | 14 |

### 15.4 繪製規則

- **可見面判斷**：以 shoelace 法計算外側面投影帶號面積（`sArea`）；面積 < 0 → 外側面朝向視線（`isOuter=true`），否則內側面（`isOuter=false`）。
- **深度排序**：entry depth = 可見面 depth − 0.001，繪於該面之上（Painter's Algorithm）。
- **僅繪可見面**；不繪背向視線那面。
- **折疊段連動**：折疊段孔跟隨「摺疊段」開關；主板孔獨立受「圓孔」開關控制。
- **符號**：實心圓點，`rgba(255,255,255,0.80)`。
- **半徑**：`max(HOLE_R_M × S, 1.5 × dpr)`。

### 15.5 顯示開關

| 開關 ID | 參數鍵 | 預設 |
|---------|--------|------|
| `#togScrewHoles` | `showScrewHoles` | on |

- **2D 模式下強制隱藏**：渲染條件為 `show3D && showScrewHoles`；切換至 2D 時圓孔不顯示，`showScrewHoles` 值保持不變，切回 3D 後自動還原。

### 15.6 程式碼常數

```js
HOLE_R_M = 0.006   // 孔半徑（m）
SCREW_Z  = [0.23, 0.43, …, 3.43]   // 17 個 z 位置
SCREW_U  = [0.10, 0.30, …, 2.90]   // 15 個 u 位置
CELL_Z   = [0.33, 0.53, …, 3.33]   // 16 個格心 z
CELL_U   = [0.20, 0.40, …, 2.80]   // 14 個格心 u
```
