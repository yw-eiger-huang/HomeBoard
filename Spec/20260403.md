## 6. 投影規格

### 6.1 Cabinet Oblique 基本參數

```
投影角 30°，foreshortening 0.5×

OBL_COS = cos(30°) × 0.5 ≈ 0.4330
OBL_SIN = sin(30°) × 0.5 = 0.25
```

### 6.2 水平視角旋轉（Y 軸旋轉，v3.1）

視角旋轉角 φ（`params.hViewAngle`，0°～90°）繞 Y 軸旋轉整個場景：

```
cosV = cos(φ),  sinV = sin(φ)

h(x,z) = x·cosV + z·sinV        ← 水平螢幕分量
d(x,z) = −x·sinV + z·cosV       ← 深度（轉為斜平行偏移）

p(x, y, z):
  canvas_x = OX + h·S + d·S·OBL_COS
  canvas_y = OY − y·S − d·S·OBL_SIN
```

- φ=0°：h=x, d=z → 側視圖，與 v3.0 完全一致。
- φ=90°：h=z, d=−x → 正視圖，板寬水平展開，板長退入深度。
- 所有 canvas 繪圖呼叫統一使用 `p(x,y,z)→[canvas_x,canvas_y]`。

### 6.3 動態場景邊界

```
場景四角 (x,z) ∈ {0, AX_W+T+0.5} × {0, BOARD_W}
  hc = h(x,z) + d(x,z)·OBL_COS
minH = min(hc),  maxH = max(hc)
sceneW = maxH − minH + 0.2
OX = margin + viewOX + 44·dpr − minH·S    ← 場景左端對齊留白
```

> 任意 φ 下場景均不被裁切。

## 8. 剛性桿 3D 繪製

### 8.1 位置

H-M-G 桿組各繪製兩組：
- **後組**：z = BOARD_W
- **前組**：z = 0

三個端點（H、M、G）在 z=0 與 z=BW 之間各加一條深度連接線（連接前後對應點）。

### 8.2 繪製順序

1. 深度連接線（H/M/G 各點前後連線），`#90d8c055`，lineWidth=1.5
2. 後組桿（z=BW），圓柱三層描線，較暗色
3. 前組桿（z=0），圓柱三層描線，標準色（含長度標籤）

### 8.3 圓柱三層描線規格

```
每段桿（H→M 或 M→G）：
  計算桿方向在 canvas 的法向量 (px, py)
  offset = 1.5px

  shadow（外輪廓）：strokeStyle=#508070，lineWidth=2×dpr，偏移 +offset
  body（主體）：    strokeStyle=#90d8c0，lineWidth=8×dpr（前組）/ 6×dpr（後組）
  highlight（反光）：strokeStyle=#c0f0e8，lineWidth=2.5×dpr，偏移 −offset
  lineCap = 'round'
```

後組較暗色：body=#6aac9a，shadow=#3a6d5a，highlight=#8ecfbc。

### 8.4 標籤

長度標籤（cm）只繪製在前組（z=0）的 H-M 和 M-G 各中點，樣式同 v2.6。
