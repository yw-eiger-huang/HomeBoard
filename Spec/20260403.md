## 1. 基本構型描述

承 v2.6。新增第三軸（Z = 板寬/深度），使木板與折疊段呈現 3D 矩形柱體；H-M-G 剛性桿呈現圓柱棍狀物，前後各一組。核心幾何（§5）仍為 2D 側視圖計算，Z 軸僅用於投影繪製。v3.1 新增水平視角旋轉（φ，§6.2）；v3.2 新增 3D/2D 切換開關、深度排序繪製清單、內側面；v3.3 切換 2D 時自動重設 φ=0°，切換回 3D 時還原上次角度；v3.4 水平視角旋轉改以場景水平中心為旋轉軸（§6.3）。

## 6. 投影規格

### 6.1 Cabinet Oblique 基本參數

```
投影角 30°，foreshortening 0.5×

OBL_COS = cos(30°) × 0.5 ≈ 0.4330
OBL_SIN = sin(30°) × 0.5 = 0.25
```

### 6.2 水平視角旋轉（Y 軸旋轉）

視角旋轉角 φ（`params.hViewAngle`，−90°～90°）繞 Y 軸旋轉整個場景：

```
cosV = cos(φ),  sinV = sin(φ)

h(x,z) = x·cosV + z·sinV        ← 水平螢幕分量
d(x,z) = −x·sinV + z·cosV       ← 深度（轉為斜平行偏移）

p(x, y, z):
  canvas_x = OX + h·S + d·S·OBL_COS
  canvas_y = OY − y·S − d·S·OBL_SIN
```

- φ=0°：h=x, d=z → 側視圖，與 v3.0 完全一致。
- φ=±90°：板寬水平展開，板長退入深度。
- 所有 canvas 繪圖呼叫統一使用 `p(x,y,z)→[canvas_x,canvas_y]`。
- 水平視角滑桿僅在 3D 模式下顯示。
- **模式切換與角度記憶**：切換至 2D 時，自動將 φ 重設為 0° 並同步更新滑桿；切換回 3D 時，還原切換前的 φ 值（儲存於 `savedHViewAngle`）。

### 6.3 場景縮放

```
sceneW = AX_W + T + 0.5    ← 固定 2D 公式，φ=0 時與原 2D 側視圖等比例
sceneH = CEIL_H × 1.22
S      = min((W−2·margin)/sceneW, (H−2·margin)/sceneH) × viewScale
```

**中心旋轉軸**（v3.4）：OX 以場景 XZ 中心點 `(cx, cz) = ((AX_W+T+0.5)/2, BOARD_W/2)` 在 φ=0 時的投影水平座標為固定基準：

```
hcCenter0 = cx + cz·OBL_COS                              ← φ=0 時的水平分量
hcCenter  = (cx·cosV + cz·sinV) + (−cx·sinV + cz·cosV)·OBL_COS
OX        = margin + viewOX + 44·dpr + (hcCenter0 − hcCenter)·S
```

旋轉任意 φ 時，`(OX + hcCenter·S)` 保持不變，場景在畫布上原地旋轉，不左右偏移。

## 7. 板面繪製（3D 模式）

### 7.1 深度排序清單（v3.2）

所有可繪元素（板面、內側面、Ghost C 延伸、中心線、剛性桿段）統一加入 `drawList`，以每個元素的平均深度 `d̄ = mean(−x·sinV + z·cosV)` 由大到小排序後繪製（Painter's Algorithm）。

## 8. 剛性桿繪製

### 8.1 3D 模式

- **遠組**（depth 較大的 z 端）：圓柱三層描線，較暗色（body=#6aac9a，6×dpr）
- **近組**（depth 較小的 z 端）：標準色（body=#90d8c0，8×dpr）；含長度標籤
- 深度連接線（H/M/G 各點前後連線）：`#90d8c055`，lineWidth=1.5
- 遠近判斷：`cosV ≥ 0` → z=BW 較遠；`cosV < 0` → z=0 較遠
- 所有桿段加入 `drawList` 統一深度排序

### 8.2 2D 模式

只繪製近組（z=0 那側）單組桿，無深度連接線。

### 8.3 圓柱三層描線規格

```
shadow：strokeStyle=暗色，lineWidth=2×dpr，偏移 +1.5px（法向量方向）
body：  strokeStyle=主色，lineWidth=8×dpr（近）/ 6×dpr（遠）
highlight：strokeStyle=亮色，lineWidth=2.5×dpr，偏移 −1.5px
lineCap = 'round'
```
