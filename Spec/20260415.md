## 18. 邊框標示（Border Mark）

### 18.1 功能概述

在主板（A→B）與摺疊板（B→D）的大面板四邊往內標示出一個較深顏色的填色區域，協助施工時確認安全邊距。

### 18.2 顯示條件

| 開關 ID | 參數鍵 | 預設 |
|---------|--------|------|
| `#togBorderMark` | `showBorderMark` | on |

關閉時，兩個寬度 slider row 一併隱藏（`display: none`）。

### 18.3 邊框寬度參數

| 滑桿 ID | 參數鍵 | 範圍 | 預設 | 作用邊 |
|---------|--------|------|------|--------|
| `#borderMarkWidth` | `borderMarkWidth` | 3.6–10 cm | 5 cm | A 端、D 端、左右 Z 邊 |
| `#borderMarkJunctionWidth` | `borderMarkJunctionWidth` | 3–5 cm | 5 cm | B 交接端（主板 B 端 + 摺疊板 B 端） |

Step 均為 0.1 cm；值以公尺儲存（`cm / 100`）。

### 18.4 幾何定義

設 `M = borderMarkWidth`，`J = borderMarkJunctionWidth`。

#### 正面視圖（Y-Z 投影，`frontview.js`）

**主板**（大面尺寸 MAIN_LEN × BOARD_W）：

```
內框上邊  mT = AY_w − M × cos(θ)     // A 端往內 M
內框下邊  mB = BY_w + J × cos(θ)     // B 端往內 J
內框左邊  z  = M                      // Z=0 邊往內 M
內框右邊  z  = BOARD_W − M            // Z=BW 邊往內 M
```

**摺疊板**（大面尺寸 FOLD_LEN × BOARD_W）：

```
內框上邊  fT = BY_w + J × fdy         // B 端往內 J（沿折疊方向）
內框下邊  fB = DY_w − M × fdy         // D 端往內 M
內框左邊  z  = M
內框右邊  z  = BOARD_W − M
```

#### 3D 視圖（`draw.js`）

在可見大面（外側面或內側面，由 shoelace 法判斷）繪製外框 minus 內框（evenodd fill）：

- topMarg / botMarg 對應沿板方向兩端各自的邊距（主板：top=M, bot=J；摺疊板：top=J, bot=M）
- Z 方向兩邊固定使用 M

#### 側視圖 2D（`draw.js`，`show3D=false`）

在板端截面繪製填色四邊形：

| 位置 | 寬度 |
|------|------|
| 主板 A 端 | M |
| 主板 B 端 | J |
| 摺疊板 B 端 | J |
| 摺疊板 D 端 | M |

### 18.5 視覺樣式

| 屬性 | 值 |
|------|-----|
| 填色 | `rgba(0,0,0,0.35)`（暗色疊加，使邊框區域呈較深同色調） |
| 繪製方式 | `ctx.fill('evenodd')`（外框路徑 + 內框路徑組合） |
| 深度排序（3D） | 可見面 depth − 0.002，確保繪於板面之上 |
| 線型 | 無邊線，純填色 |
