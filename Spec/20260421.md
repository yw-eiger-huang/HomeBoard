## 22. 板厚與框架厚度常數

| 常數 | 值 | 說明 |
|------|----|------|
| `T` | 0.018 m（1.8 cm） | 面板厚度（外側面） |
| `T_FRAME - T` | 0.027 m（2.7 cm） | 邊框深度（內側背面） |
| `T_FRAME` | `T + 0.027` = 0.045 m（4.5 cm） | 組合總深（面板 + 邊框） |
| `HT` | `T_FRAME / 2` | 截面中心線偏移 |

A、G、B 鉸接點位於外側面（距內面 |T_FRAME| 處）。所有外側邊點（`Ao, Bo, Co, Bfo, Do`）以 `T_FRAME × 法向量` 從內側邊偏移。

## 23. 板料構造（面板 + 邊框）

板子採「面板在前、邊框在後」構造：

- **面板**：厚度 |T| = 1.8 cm，位於外側面（朝房間），全面完整拼接，無邊框外露。
- **邊框**（外框與內部分隔條）：深度 |T_FRAME-T| = 2.7 cm，固定於板面後方（內側/靠牆側）。
- A、G、B 鉸接點位於最外側面（面板外表面）。
- 邊框可見寬度（面板面）由 `borderMarkWidth` 控制，預設 |T_FRAME| = 0.045 m。
- B 交接端寬度由 `borderMarkJunctionWidth` 控制，預設同 `borderMarkWidth`。

## 24. 分割帶常數與渲染

### 24.1 常數（`constants.js`）

```js
MAIN_DIV_WIDTHS = [0.48, 0.90, 0.90, 0.90, 0.48]  // 主板各欄寬（Z 軸，m）
FOLD_DIV_WIDTHS = [1.18, 1.30, 1.18]               // 摺疊板各欄寬（Z 軸，m）
```

分割線 Z 位置 = 各欄寬累積和（去掉末位）。

### 24.2 正面視圖

同 §19，改用 `MAIN_DIV_WIDTHS`、`FOLD_DIV_WIDTHS` 常數，行為不變。

### 24.3 3D 視圖

`showBorderMark = true` 時，在可見大面繪製分割帶：

| 屬性 | 值 |
|------|----|
| Z 範圍 | `[z_div − MARGIN/2, z_div + MARGIN/2]` |
| 長度方向 | topPt → botPt（板全長） |
| 填色 | `rgba(0,0,0,0.35)` |
| 深度 | 可見面 depth − 0.002 |
| 數量 | 主板 4 條、摺疊板 2 條 |

## 25. 雙層厚度標註（側視圖，`showDims = true`）

B 點處以共用線段（`Bw → Bo`）繪製兩層標註；另加面板分界虛線貫穿全板長：

### 25.1 B 點標註

| 標註 | 位置 | 文字 |
|------|------|------|
| 邊框（內側） | `Bw`（內側面） | `框 2.7cm` |
| 面板（外側） | `Bo`（外側面） | `板 1.8cm` |

- 共用線：`Bw → Bo`（總深 4.5cm）
- 加重線段：`Bw + (T_FRAME-T) × 法向量 → Bo`（標示 1.8cm 面板層）
- 字型 `FS.dim−1`，顏色 `GRAY_DASH+'88'`，對齊 `left`，偏移 `+8dpr`。

### 25.2 面板分界虛線

沿板全長繪製虛線，位於距內側面 `T_FRAME-T = 2.7cm`（即距外側面 T = 1.8cm）：

| 屬性 | 值 |
|------|----|
| 主板 | `Aw + (T_FRAME-T)×法向量 → Bw + (T_FRAME-T)×法向量` |
| 摺疊段 | `Bw + (T_FRAME-T)×折疊法向量 → Dw + (T_FRAME-T)×折疊法向量` |
| 3D 模式 | z=0 及 z=BW 各繪一條 |
| 線型 | `GRAY_DASH+'55'`，虛線 `[3,3]`，受 `showDims` 控制 |

## 26. 主板中段水平框架（背面橫條）

### 26.1 位置

框架中心位於主板長度的中點：

```
u_nominal = (FOLD_LEN + BOARD_LEN) / 2   // 從 D 端量起
```

若中心附近有圓孔，向 A 端偏移直至淨空：

```
while (∃ hu ∈ holes : |hu − u_frame| < frameH/2 + HOLE_R_M)
    u_frame += 0.005   // 每步 0.5 cm，上限 BOARD_LEN − frameH
```

其中 `holes` = `SCREW_U ∪ CELL_U`（主板段，u ≥ FOLD_LEN）。

### 26.2 尺寸

| 屬性 | 值 |
|------|----|
| 高度（沿板方向） | `borderMarkWidth`（同邊框） |
| 寬度（Z 軸） | `BOARD_W`（全寬） |

### 26.3 渲染

#### 正面視圖

- **填色帶**（`showBorderMark`）：`rgba(0,0,0,0.35)`，全寬，高度 = `frameH × cos(θ)`（投影）
- **中心虛線**：與縱向分割線相同樣式（`rgba(196,137,74,0.45)`，`[5,4]` dash），橫跨全板寬

#### 側視圖（2D）

在板截面中點繪製 `rgba(0,0,0,0.35)` 填色矩形，頂點為：
- `Aw + (dist ± frameH/2) × (ddx, ddy)` 靠牆側
- `Ao + (dist ± frameH/2) × (ddx, ddy)` 外側
- `dist = BOARD_LEN − u_frame`

#### 3D 視圖

在可見面（外側或內側，依視角）繪製同尺寸填色矩形帶，Z 從 0 到 `BW`。

### 26.4 尺寸標註（`showDims`，正面視圖）

在板左側繪製垂直標註線，標示 A 端至框架中心距離：

| 屬性 | 值 |
|------|----|
| 位置 | `cvX(0) − dimOff` |
| 範圍 | `cvY(AY_w) → cvY(mfHy)` |
| 標籤 | `(mfDist × 100).toFixed(0) + 'cm'` |
