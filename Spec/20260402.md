# 木板構型幾何規格文件

> 版本：2.6 | 座標系：2D 側視圖 | 單位：公尺（除非特別標註）

---

## 1. 基本構型描述

一塊木板以單點 hinge（A 點）固定於天花板，木板**外傾**（底端朝牆方向斜下），底端在設計角度（40°）時剛好接觸牆面。木板底部 65 cm 為可折疊段，以 hinge 繞 B 點旋轉，摺疊方向僅限朝牆面方向（往內折）。**摺疊角度無 UI 控制，隨牆面夾角自動同步。**

## 4. 幾何關鍵點定義

所有坐標以**靠牆側邊緣**為基準（含 G、M）。A、H、I 為固定點；其餘點隨 θ 變動。

| 點 | 名稱 | 說明 |
|----|------|------|
| **A** | Hinge | 天花板固定點，靠牆側邊緣 |
| **B** | 折點 | 主板底端 = 摺疊段頂端，靠牆側邊緣，摺疊旋轉軸 |
| **C** | 主板底端 | 木板全長末端（未折疊位置），靠牆側邊緣 |
| **D** | 摺疊端 | 摺疊段末端，靠牆側邊緣 |
| **G** | 剛性桿接點 | 主板靠牆側邊緣上，距 A 點 `g`（可調）之點 |
| **H** | 牆角 | 天花板與牆面交點 (0, 2.5) |
| **I** | 全長觸牆點 | θ=40° 時 C 點位置（觸牆參考點） |
| **M** | 剛性桿交界點 | H-M、M-G 兩段等長剛性桿的交界點；長度以 θ=0° 時 H→G 中點計算，固定後隨 G 移動而彎折朝向 A |

## 5. 座標計算公式

### 5.1 輸入變數

| 變數 | 範圍 | 說明 |
|------|------|------|
| `θ`（angle） | 0° ~ 40° | 木板靠牆側邊緣與牆面的夾角；0° = 垂直貼牆，40° = 設計角度 |
| `g`（gOffset）| 1.75 ~ 2.15 m | G 點距 A 點沿板邊距離（使用者可調） |

### 5.2 方向向量

```
dir      = (−sin θ, −cos θ)              ← 主板方向
norm     = (+cos θ, −sin θ)              ← 主板外法向量（遠離牆側）
boardDir = atan2(−cos θ, −sin θ)
```

### 5.3 靠牆側邊緣座標（inner edge）

```
A = (1.9593,  2.500)                               ← 固定
B = A + 2.398 × dir
C = A + 3.048 × dir
G = A + g × dir                                    ← 隨 gOffset 移動
H = (0, 2.500)                                     ← 固定
I = (0, 2.500 − 3.048 × cos 40°) ≈ (0, 0.165)    ← 固定
```

### 5.4 摺疊段（自動同步）

```
foldRad  = (2π/3) × (1 − θ/40°)         ← θ=40° 時 0（D=C），θ=0° 時 120°
foldDir  = boardDir − foldRad
fold_dir  = (cos foldDir, sin foldDir)
fold_norm = (−sin foldDir, cos foldDir)  ← 摺疊段外法向量

D = B + 0.650 × fold_dir
```

### 5.5 板厚偏移點

```
── 主板外側邊緣（+ norm × 0.05）──
A_outer      = A + 0.05 × norm
B_outer      = B + 0.05 × norm
C_outer      = C + 0.05 × norm

── 摺疊段外側（+ fold_norm × 0.05）──
B_fold_outer = B + 0.05 × fold_norm
D_outer      = D + 0.05 × fold_norm

── 主板中心線（+ norm × 0.025）──
A_cen = A + 0.025 × norm
B_cen = B + 0.025 × norm
```

### 5.6 剛性桿交界點 M（兩等長桿，動態長度）

**桿長（隨 gOffset 動態計算，每次重算）：**
```
LEN_HM = LEN_MG = 0.5 × √(AX_W² + g²)
```
> 此為 θ=0° 時 H→G 連線的一半，即中點距離；g 改變時桿長同步更新。

**任意 θ 時：**
```
d    = |H − G|
a    = d / 2                                        ← 兩段等長時
h    = √(LEN_HM² − a²)
pm   = H + a × (G − H) / d                         ← H→G 連線上的基點
perp = (−(G.y − H.y), G.x − H.x) / d              ← H→G 的左法向量
cross = (G − H) × (A − H)                          ← z 分量：A 在 H→G 的哪一側

若 cross ≥ 0：M = pm + h × perp
否則：        M = pm − h × perp
```

> θ=0° 時 h=0，M = pm（無彎折）；θ 增大時 M 朝 A 方向彎折。

## 6. 板面多邊形（繪製用）

```
主板（四邊形）：    A → B → B_outer → A_outer
摺疊段（四邊形）：  B → D → D_outer → B_fold_outer
```

> θ=40° 時 foldRad=0，`B_fold_outer` 與 `B_outer` 重合；其他角度不重合。

## 9. 輔助量測

| 量測項 | 公式 |
|--------|------|
| H→G 距離 | `\|H − G\|` |
| H→B 距離 | `\|H − B\|` |
| M→A 距離 | `\|M − A\|` |
| A→B 水平投影 | `A.x − B.x` |
| B 離地 | `B.y` |
| C 離地 | `C.y` |
| 內縮基準點離地高度 | 見 §9.1 |

### 9.1 內縮基準點投影高度

使用者可調整「內縮距離」d（100～150 cm），以 Ao（主板外側頂角）的地面投影為基準往牆面方向內縮 d，向上投影至主板外側面，標示交點離地高度。

```
refFloorX = Ao.x − d
t         = (refFloorX − Bo.x) / (Ao.x − Bo.x)    （t ∈ [0,1] 時有效）
projY     = Bo.y + t × (Ao.y − Bo.y)
```

> t ∉ [0,1] 時（主板 x 範圍不涵蓋 refFloorX）標示自動隱藏。

**視覺元素（「標註尺寸」開啟時）：**
- 地面水平 dim 線：refFloorX → Ao.x 地面投影，標示 d
- 垂直灰色虛線：(refFloorX, 0) → (refFloorX, projY)
- 交點灰色圓點；高度標籤（cm）顯示在虛線右側中點

## 12. 響應式介面（RWD）規格

### 12.1 版面邏輯

| 情境 | 條件 | 面板位置 | 面板寬度 |
|------|------|----------|----------|
| 桌面版 | viewport 寬 > 640px | 右側 | 240px |
| 手機直式 | `max-width: 640px` + `portrait` | 畫布下方 | 全寬 |
| 手機橫式 | `max-height: 500px` + `landscape` | 右側（緊湊）| 180px |

手機版（直式與橫式）字體縮減：label 11px、stat 10px。

直式：主佈局改垂直 flex，面板 `width:100%` + `border-top`，section 水平 flex wrap。
橫式：沿用水平 flex，section padding 縮減為 8px 10px。

### 12.2 Canvas 繪圖縮放與定位

```
margin = min(90×dpr,  min(W,H) × 0.025)
sceneW = AX_W + T + 0.5 = 2.509 m
sceneH = CEIL_H × 1.22  = 3.050 m
autoS  = min((W − 2×margin) / sceneW,  (H − 2×margin) / sceneH)
S      = autoS × viewScale

OX = margin + viewOX + 44×dpr   ← 牆面左側 annotation 留白
OY = margin + CEIL_H×S + viewOY ← 天花板錨定於頂部 margin
```

- 初始（viewScale=1）場景填滿 canvas 短邊的 **95%**
- 天花板線始終錨定於 canvas 頂部 margin

### 12.3 觸控操作

| 手勢 | 動作 |
|------|------|
| 單指拖曳 | 平移（pan） |
| 雙指捏合 / 張開 | 縮放（pinch zoom），範圍 0.15 ~ 12× |

所有 touch 事件呼叫 `preventDefault()`（`passive: false`）。
hint 文字：`拖曳平移 · 滾輪縮放 · 雙指縮放`
