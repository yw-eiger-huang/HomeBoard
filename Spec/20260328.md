# 木板構型幾何規格文件

> 版本：2.2 | 座標系：2D 側視圖 | 單位：公尺（除非特別標註）

---

## 1. 基本構型描述

一塊木板以單點 hinge（A 點）固定於天花板，木板**外傾**（底端朝牆方向斜下），底端在設計角度（40°）時剛好接觸牆面。木板底部 65 cm 為可折疊段，以 hinge 繞 B 點旋轉，摺疊方向僅限朝牆面方向（往內折）。**摺疊角度無 UI 控制，隨牆面夾角自動同步。**

## 5. 座標計算公式

### 5.1 輸入變數

| 變數 | 範圍 | 說明 |
|------|------|------|
| `θ`（angle） | 0° ~ 40° | 木板靠牆側邊緣與牆面的夾角；0° = 垂直貼牆，40° = 設計角度 |

> 摺疊角度無獨立輸入，由 θ 自動推算（見 5.4）。

### 5.2 主板方向向量

```
主板方向向量：  dir  = (−sin θ, −cos θ)
主板外法向量：  norm = (+cos θ, −sin θ)   ← 垂直板面，指向遠離牆側
boardDir = atan2(−cos θ, −sin θ)
```

### 5.3 主板靠牆側邊緣座標

```
A = (AX,  AY)                      AX = 3.048 × sin(40°) ≈ 1.9593 m
                                    AY = 2.500 m

B = A + 2.398 × dir
C = A + 3.048 × dir
```

### 5.4 摺疊段方向（自動同步）

摺疊方向往內側（朝牆面），隨牆面夾角線性同步：

```
foldRad = (2π/3) × (1 − θ/40°)    ← θ=40° 時 foldRad=0（D=C），θ=0° 時 foldRad=120°

foldDir = boardDir − foldRad

fold_dir = (cos foldDir, sin foldDir)
fold_norm = (−sin foldDir, cos foldDir)   ← 摺疊段外法向量（+90° 旋轉）

D = B + 0.650 × fold_dir
```

### 5.5 外側邊緣座標（板厚偏移）

```
A_outer = A + 0.05 × norm
B_outer = B + 0.05 × norm
C_outer = C + 0.05 × norm

摺疊段外側上角：  B_fold_outer = B + 0.05 × fold_norm
摺疊段外側下角：  D_outer      = D + 0.05 × fold_norm
```

### 5.6 中心線座標（板厚 / 2 偏移）

```
A_cen = A + 0.025 × norm
B_cen = B + 0.025 × norm
```

### 5.7 靠牆側邊緣四等分點

```
E = A + (1/4) × 2.398 × dir
F = A + (2/4) × 2.398 × dir
G = A + (3/4) × 2.398 × dir
```

### 5.8 固定參考點

```
H = (0, 2.500)
I = (0, 2.500 − 3.048 × cos(40°))  ≈ (0, 0.165 m)
```

### 5.9 剛性桿交界點 M（兩圓交點法）

M 不是 H→G 的幾何中點，而是兩段定長剛性桿 H-M、M-G 的交界點。

**參考長度（θ=0° 時計算，固定不變）：**
```
G₀ = A + (3/4) × 2.398 × (0, −1)   ← θ=0 時 G 的位置
M₀ = (H + G₀) / 2                    ← θ=0 時的中點

LEN_HM = |H − M₀|
LEN_MG = |M₀ − G₀| = LEN_HM         ← M₀ 為中點，兩段等長
```

**任意 θ 時 M 的計算（兩圓交點）：**
```
d    = |H − G|
a    = (LEN_HM² − LEN_MG² + d²) / (2d)   = d/2（兩段等長時）
h    = √(LEN_HM² − a²)
pm   = H + a × (G − H) / d                ← H→G 連線上的基點

perp = (−(G.y − H.y), G.x − H.x) / d     ← H→G 的左法向量

cross = (G − H) × (A − H)                 ← z 分量：判斷 A 在 H→G 的哪一側

若 cross ≥ 0：M = pm + h × perp           ← A 在左側，取左交點
否則：        M = pm − h × perp
```

> θ=0° 時 h=0，M = pm = H→G 中點（無彎折）。
> θ 增大時 G 靠近 H，h 增大，M 朝 A 方向彎折。

## 6. 板面多邊形（繪製用）

### 主板（四邊形）
```
頂點順序：A → B → B_outer → A_outer → (close)
```

### 摺疊段（四邊形，剛性長方形）
```
頂點順序：B → D → D_outer → B_fold_outer → (close)
```

> `B_fold_outer` 隨 `fold_norm` 旋轉，與主板外側 `B_outer` 不重合（除 θ=40° 時 foldRad=0）。

## 8. 角度約束條件

### 牆面夾角（θ）
- 範圍：**0° ~ 40°**
- 0° 極限：木板垂直貼牆（B 與 C 均在牆面上）
- 40° 極限：設計角度，C 觸牆，D=C

### 摺疊角度（自動推算）
- 由 `foldRad = (5π/6) × (1 − θ/40°)` 決定，**無獨立輸入**
- θ=40°：foldRad = 0°（D=C，完全展開）
- θ=0°：foldRad = 120°（最大內折）
- 注意：θ < ~35° 時 D.y 可能低於地面（幾何限制）

## 9. 輔助量測

| 量測項 | 公式 |
|--------|------|
| H→G 直線距離 | `√((G.x − 0)² + (G.y − 2.5)²)` |
| H→B 直線距離 | `√((B.x − 0)² + (B.y − 2.5)²)` |
| M→A 距離 | `√((A.x − M.x)² + (A.y − M.y)²)` |
| M→E 距離 | `√((E.x − M.x)² + (E.y − M.y)²)` |
| A→B 段對地面投影長度 | `A.x − B.x` |
| B 對牆面投影高度 | `B.y` |
| C 離地 | `C.y` |
